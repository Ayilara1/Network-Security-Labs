## 📖 Overview

This repository contains my hands-on **Network Security** laboratory exercises completed during my cybersecurity training.

The labs focus on networking fundamentals, network protocols, routing, VLAN configuration, IP addressing, and network security principles. Practical activities were carried out using industry-standard networking concepts and security practices.

---

# 📂 Repository Contents

| Lab | Description |
|------|-------------|
| 📘 Introduction to Network Security | Fundamental networking concepts and security principles |
| 🌍 Understanding Network Layers and TCP/IP Model | OSI Model, TCP/IP Protocol Suite and packet flow |
| 📡 Network Security Protocols and Configuration | Secure communication protocols and configuration |
| 🔐 Network Security Policies and Risk Management | Security policies, risk assessment and best practices |
| 🌐 Simulating Network Routing and VLAN Configuration | VLAN creation, routing configuration and segmentation |
| 📊 IP Address Analysis and Network Report | IPv4 subnetting, address planning and network analysis |
